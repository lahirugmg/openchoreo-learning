# Source Code Folder

This `code/` directory is reserved for local source checkouts used by this learning plan.

## Clone OpenChoreo Here

Please check out the OpenChoreo source repository into this folder:

```bash
cd /Users/lahirus/work/pet-projects/open-choreo/openchoreo-learning/code
git clone https://github.com/openchoreo/openchoreo.git
```

Expected path after cloning:

```text
code/openchoreo
```

If `code/openchoreo` already exists, update it with:

```bash
cd /Users/lahirus/work/pet-projects/open-choreo/openchoreo-learning/code/openchoreo
git fetch --all --prune
git pull --rebase
```
